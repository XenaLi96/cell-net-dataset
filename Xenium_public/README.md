# Xenium Public Block

This directory is reserved for the Xenium public data block of sMMC-22M.

After the paper is published, this block will be distributed through Globus. We will
update this directory with the Globus access information, release manifest, file
checksums, and download instructions.

The Xenium public block is expected to contain platform-defined segmented-cell
records and associated metadata for the Xenium portion of the release.
