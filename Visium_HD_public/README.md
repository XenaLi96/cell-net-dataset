# Visium HD Public Block

This directory is reserved for the Visium HD public data block of sMMC-22M.

After the paper is published, this block will be distributed through Globus. We will
update this directory with the Globus access information, release manifest, file
checksums, and download instructions.

The Visium HD public block is expected to contain bin-to-cell aligned records and
associated metadata for the Visium HD portion of the release.
