# Full Release

This directory is reserved for the full sMMC-22M release.

After the paper is published, the full release will be distributed through Globus.
We will update this directory with the Globus access information, release manifest,
file checksums, and download instructions.

The full release is expected to contain the complete aligned single-cell resource
across Xenium and Visium HD data blocks, with image, omics, metadata, spatial, and
benchmark-related artifacts.
